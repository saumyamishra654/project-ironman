# Half Marathon Plan — Shin Bridge Block

*Prepends to `HM_Plan_Delhi_Oct18`. Replaces the running load of roughly the first 3 weeks, then hands back to the main plan once the shin clears the reintroduction gate below. Everything not mentioned here (lifting, mobility, heat, phase map, Week-1 TT) stands as written in the main file.*

---

## Why this block exists

- **Symptoms:** diffuse, mid-shin ache, **post-run only, never during**, not focal on palpation, not worsening. That's the *manageable* (MTSS-ish load-intolerance) end of the spectrum, **not** the bone-stress end.
- **But:** it showed up at **week 0 — the plan's lowest-volume week.** That means current running volume already sits near the tibia's tolerance, and the main plan ramps *from there*. So we de-risk before building, rather than ramping into a shin that's already complaining.
- **Governing principle:** tibial tolerance is only ever built by graduated **impact**. Cross-training holds the aerobic engine but builds *zero* bone/tendon tolerance. So the move is not "delete running" — it's cut the *low-value* impact, keep the *high-value* impact, and rebuild by rule.
- **Net effect:** impact sessions drop **4 → 2** (long run + quality run). Everything else is zero- or low-impact.

## The reintroduction gate — add running by rule, not by feel

This is the part that quietly decides whether the block works. **Pain settling is not the signal that the bone is ready** — a tibia can feel completely fine while still under-adapted. So progress on a schedule you set in advance:

- **Symptom-free week** = a full week at the current load with **zero post-run ache.**
- After each symptom-free week, make **ONE** change — never both in the same week:
  - extend the long run by **≤ 1.5 km**, *or*
  - convert **one** cross-train session back to an easy run.
- If the ache returns → **drop back to the last symptom-free load, hold there another full week**, then retry the same step.
- **Never chase missed distance.** No catch-up long runs to "get back on schedule."
- Once you're at ~2–3 runs/week symptom-free with the long run climbing cleanly, you've rejoined the main plan — pick it up from **whatever volume you've actually reached, not the calendar week.**

## Duration & transition — how you get back to the main plan

**Short answer to "a month?": roughly, yes — plan for ~4 weeks, but it's symptom-gated, not calendar-locked.** Each step below only happens after a fully symptom-free week, and there are ~4 steps to make before you're main-plan-shaped. If a week isn't clean you repeat it, so **5–6 weeks is a normal outcome, not a failure** — don't compress it to chase Oct 18 (that's what the goal reframe is for). The bridge itself *is* your deload, so no separate down week inside it; resume the main plan's Wk-4/Wk-8 down weeks once you rejoin.

Each row = **exactly one change** from the row above, and only if the previous week had **zero post-run ache**. Ache returns → drop to the previous row, hold, retry the same step. Quality run stays **threshold-only (no VO2)** for the whole bridge.

| Week | Long run | Runs | The one change | Gate to advance |
|---|---|---|---|---|
| **B1** | 10 km | 2 (quality + long) | — (baseline: is even *this* clean?) | Symptom-free → B2. Still aching → hold, or cut further (long → 8 km / quality → easy). Focal or worsening → clinician. |
| **B2** | 11.5 km | 2 | +1.5 km long run | Symptom-free → B3 |
| **B3** | 11.5 km | 3 | Wed **bike → easy run R3** (5–6 km); long run *held* | Symptom-free → B4 |
| **B4** | 13 km | 3 | +1.5 km long run; runs *held* | Symptom-free → rejoin |
| **Rejoin** | +≤1.5 km/wk | → 4 | add back **R4** (recovery + strides) as the next change, then follow the main plan | you're now running the main plan |

**Why long-run first, runs second:** extending the long run by 1.5 km is a smaller impact step than adding a whole new running day (which takes you from 2 → 3 impact days). So you bank the small increments first and only add loading *frequency* once a couple of clean weeks are behind you. The long run is also your actual limiter (endurance), so it's the adaptation worth prioritising.

Where this lands you: ~4 clean weeks → 3 runs + a 13 km long run, which is roughly the main plan's early-block volume. You rejoin **by volume reached, not by calendar week** — likely somewhere around its Wk-3/Wk-4 shape, conveniently near a down week.

## Quality run — what to run (and what to drop)

**Retain answer: keep the *threshold* sessions, drop the *VO2* sessions, defer the max-effort TT.** It's not a clean "use the original plan as-is":

- The bridge overlaps the main plan's **threshold base phase (Wks 1–4)**, which is *already* all threshold/tempo — so those sessions are exactly the shin-appropriate ones and carry over almost unchanged.
- The main plan's **VO2 intervals** (Wk 5 onward, ~5:05–5:15/km) are the ones to park. Faster pace = higher peak impact force per stride = more tibial load. They wait until you've cleared the gate. If you rejoin around the plan's Wk 3–4, you meet VO2 on its original schedule anyway — you lose nothing.
- The **Week-1 5k/10k time trial is a max effort** — a concentrated impact stressor you don't want on a symptomatic shin. Defer it. Calibrate off your existing **5k (~30 min) / 10k (~1:04)** for now, and run the TT as your first quality session *after* the shin's quiet.

**During the bridge, hold ONE threshold session steady — don't progress it week to week:**

- **Session:** `4 × 6 min @ threshold, 2 min jog between.` Bookend with ~1.5 km easy jog warm-up + cool-down. ≈ 7 km total, no strides.
- **Pace:** threshold = your **5:55–6:05/km** zone, run at the **controlled end (~6:05)** — "comfortably hard," never grinding. If it's a grind, you're too fast; back off.
- **Why held flat:** the gate moves **one** lever per week (long-run km, then run count). If the quality run also crept up, you'd have a hidden second change confounding whether an increment was actually tolerated. Keeping it steady is what makes the gate readable.
- **Structure swaps are a mild progression, not a freebie:** once `4 × 6` feels easy you can run the same ~24 min of work with fewer breaks (`3 × 8 min` or `2 × 12 min`) — but only on a symptom-free week, and treat it as *that week's one change* (i.e. don't also bump the long run that week).

**Reconnecting:** on rejoin you pick the main plan's quality progression straight back up — its Wks 1–4 are threshold (seamless), and **VO2 re-enters on the original schedule from Wk 5**. The threshold sessions also double as soft calibration: if 5:55–6:05 feels much easier or harder than "comfortably hard," nudge your zones — that's most of what the TT would've told you, without the max effort.

## Tripwires — stop and get assessed (non-negotiable)

The whole bridge assumes the benign picture. It flips to bone-stress protocol if **any** of these appear:

- Pain turns **focal / pinpoint** — a spot you can cover with a fingertip that hurts to press.
- Pain shows up **during** the run, or worsens as the run goes on, instead of only afterward.
- Pain at **rest or at night**, or worsening **week over week** despite backing off.
- **Single-leg hop** on the leg reproduces focal pain.

→ Stop running, cross-train only in ways that are fully pain-free, and see a sports-medicine doctor or physio. Note: an **early X-ray often reads normal** in the first 2–3 weeks — **MRI is the sensitive test**. Don't "test" the shin with a short run to see how it feels; that's the exact behaviour that turns a stress reaction into a fracture. *(Not medical advice — I'm working off your description; a clinician puts hands on the leg.)*

## Cadence — cheap insurance

Keep easy-run cadence **≥ 170** and watch for **overstriding** (foot landing well ahead of your centre of mass, near-straight knee). Overstriding both spikes tibial impact *and* overloads the tibialis anterior — the exact tissues in play. This is a **per-pace** correction: it matters most on the slow runs, where you're most likely to reach and overstride.

## Daily prehab additions

On top of the main plan's 10-min mobility routine, every day:

- **Tib raises** — 2–3 × 15–25 (wall tib raises or seated banded dorsiflexion). Direct strength for the tibialis anterior, i.e. the muscle behind your front-of-shin overload. Low-load, high-frequency — daily is fine and is where this one earns its keep.
- **Airplane clocks** — 1–2 sets/leg. *Airplanes* (single-leg RDL hinge, arms out like wings, slow and controlled) + *clock reaches* (balance on one leg, tap the free foot to 12 / 3 / 6 / 9 o'clock). Builds single-leg stability, hip control and ankle proprioception — the mechanics that quietly govern how much load reaches the shin. Zero impact, safe daily.
- **Banded clamshells (left-biased)** — 2 × 15–20/side, left leg can go to 2 × 20–25 or add band tension once easy. Direct, isolated glute med work for the confirmed weak left hip — the once-weekly single-leg glute bridge on leg day is a good start but sparse for a chronic asymmetry; this closes the gap on off days. Zero impact, safe daily.
- **Spot jumps** — ⚠️ **this one is impact, so it's conditional, not "from day one."** The rationale is real: small, frequent impact is exactly how a tibia builds load tolerance (bone adapts to being loaded). But that only *helps* once the acute ache has settled — adding daily jumping to a currently-grumpy shin works directly against the whole block. So: **hold until you've had your first fully symptom-free week (B1 clear), then start tiny** — 2 × 10 easy pogos on the spot, soft knees, quiet landings — and progress slowly. **Stop immediately if they reproduce any shin ache**, and they're covered by the same tripwires as running.

The first three you can start now. Spot jumps wait for the shin to go quiet first — I'm deliberately not letting an impact drill sneak past the "cut impact" logic of the block.

## Thursday leg day (~65–70 min)

Pulled in from the main plan so this file stands alone. One leg day/week — enough to maintain strength and build shin/tendon capacity, light enough to leave the two runs intact. **Loads stay at RPE 7–8 — leave 2 reps in the tank.** Not a strength-PR block; grinding here taxes Sunday's long run.

**Bridge note:** with running cut to 2 sessions, your legs are actually *fresher* than on the full plan, so holding this whole session is easy to justify — and its calf/tib work is doing double duty as shin prehab. The anchor rule is unchanged: **judge the session by how the long run feels, not by how the next day feels.**

| # | Exercise | Sets × Reps | Notes |
|---|---|---|---|
| 1 | Zercher squat | 3 × 6–8 | Main squat. If forearms/biceps give out before legs, swap to front squat or leg press. |
| 2 | Barbell RDL | 3 × 8 | Heaviest posterior movement — where hamstrings actually get loaded. |
| 3 | Bulgarian split squat | 3 × 8/leg | Knee-dominant unilateral. |
| 4 | Single-leg RDL | 3 × 8/leg | Hip-dominant unilateral — every week, no alternation. |
| 5 | Sissy squat | 2–3 × 8–12 | Tibially a non-issue (no impact). Patellar issue resolved, so run it normally — but reintroduce progressively (it's patellar-tendon-intensive) and treat the knee as the canary: first to regress if that tendon ever murmurs again. |
| 6 | Leg curl (seated if available) | 2–3 × 10–12 | Knee-flexion hamstring function. Seated > lying. Nordics deferred to post-race (DOMS wrecks long runs mid-block). |
| 7 | Straight-leg calf raise | 3 × 10–12, slow 3s lowering | Gastroc. Add load once 12 is easy. |
| 8 | Bent-knee (soleus) calf raise | 3 × 12–15 | **The shin-protection exercise — soleus eats the biggest share of tibial load in running. Do not skip.** |
| 9 | Tibialis raises (loaded) | 3 × 15–20 | The *loaded* version — complements the light daily tib raises above, doesn't replace them. |
| 10 | Copenhagen plank | 2 × 20–30s/side | Adductors — most-neglected running muscle. Short lever first. |
| 11 | **Single-leg glute bridge, paused** | 2–3 × 12–15/leg, left first, 2s pause at top | **New — added for the confirmed rightward pelvic drop / weak left hip.** Reinforces holding the pelvis level in single-leg stance, the exact control that's missing. Left leg does the same reps as right despite being weaker — don't secretly shortchange it. |

**Glute med — why it's here and not just "more single-leg work":** the confirmed pattern is a **rightward pelvic drop, weak left hip** — specifically a glute *med* (lateral hip stabiliser) deficit, a different muscle from the glute *max* that #4's single-leg RDL already trains. Uncontrolled, it lets the left leg drift into adduction/internal rotation on stance, which cascades into more pronation and overloads the medial foot on that side — the mechanism behind your recurring left-arch/FHL pain, most exposed by hard toe-off on inclines like yesterday's hike.

Balance check: 4 posterior (RDL, SL RDL, leg curl, Copenhagen) vs 3 anterior (Zercher, split squat, sissy) — deliberately posterior-tilted, since hamstrings do heavy work in late swing and are the classic undertrained running muscle.

- **If the long run degrades two weeks running,** cut in this order: leg-curl sets → RDL to 2 sets. **Never cut 7 / 8 / 9** — that's injury-prevention work, not volume.
- **Fatigue deload:** the bridge is already a *running* deload, so leave this at full unless leg fatigue genuinely builds. If it does, drop to exercises **1, 2, 7, 8, 9 at 2 sets each** (the same reduction the main plan uses for down weeks/taper). The calf/tib block continues regardless.

## Swim — train it so it isn't junk volume

Your 400 m ceiling is almost certainly **breathing / pacing, not aerobic base** — you run continuously for an hour, so the central engine is clearly big enough; whatever stops you at 400 m is swim-specific.

- **Slow-swim test (next session):** swim deliberately, *annoyingly* slow, with a breathing rhythm you can hold comfortably. If distance-before-stopping jumps sharply, the limiter was pace/breathing — confirmed. Note *what* makes you stop: **air hunger** → breathing pattern; **whole-body fatigue** → closer to genuinely metabolic. (Betting heavily on the former.)
- **Progress continuous unbroken distance** at that slow pace: 500 → 600 → 800. That's what dissolves the ceiling.
- Fragmented hard 100 m sets feel productive and build the least of what you need.
- Purpose of these swims: **HIM prep** (1.9 km continuous) + shin-safe aerobic movement + adherence — **not** October HM fuel. Hold that model so you train them right.
